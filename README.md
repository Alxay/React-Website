# React-Website

To jest przykładowa strona internetowa napisana w React.

## Demo

[Zobacz stronę na żywo](https://alxay.ninja/)

## Technologie

- React
- JavaScript
- CSS

## Jak uruchomić projekt?

1. Sklonuj repozytorium:
    ```bash
    git clone https://github.com/Alxay/React-Website.git
    ```
2. Przejdź do folderu projektu:
    ```bash
    cd React-Website
    ```
3. Zainstaluj zależności:
    ```bash
    npm install
    ```
4. Uruchom aplikację:
    ```bash
    npm start
    ```

Aplikacja wystartuje pod adresem http://localhost:3000
